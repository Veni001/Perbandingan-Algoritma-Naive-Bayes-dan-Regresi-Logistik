# Perbandingan Algoritma Naive Bayes dan Regresi Logistik
Proyek ini merupakan analisis Big Data yang memanfaatkan data berskala besar dari platform Tokopedia untuk melakukan klasifikasi menggunakan PySpark. Dua algoritma machine learning, yaitu Naïve Bayes dan Regresi Logistik, diterapkan untuk mengevaluasi performa model dalam menangani data berukuran besar. Hasil analisis menunjukkan bahwa Regresi Logistik memiliki akurasi paling tinggi, melampaui Naïve Bayes, sehingga lebih efektif untuk kebutuhan klasifikasi skala besar. Proyek ini memberikan landasan bagi pengembangan sistem klasifikasi otomatis guna mendukung pengambilan keputusan di industri e-commerce.
